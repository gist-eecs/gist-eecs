## GIST EECS

**GIST EECS**는 GIST 학부 전기전산컴퓨터 전공 학생들을 필두로 IT 학습과 관련 업계 취업준비에 관심이 있는
학생들이 모여 정보를 나누는 공동체입니다.

### 활동 개요

 * 취업 공고 모음
 * IT 학습 방향 토론
 * 포트폴리오 공유
 * 모임 내용 공유

모든 활동은 [행동지침](CODE_OF_CONDUCT.md)을 따릅니다.
